# 03 - User Stories

**Project:** Mizu Care
**Version:** 1.0

---

# Purpose

This document describes how each type of user interacts with the application.

Each story represents a complete feature from the user's perspective and includes acceptance criteria that define when the feature is considered complete.

---

# Epic 1 – View Today's Schedule

## User Story

As a family member,

I want to immediately see everything that needs to be done today,

so I know what responsibilities are coming up.

### Acceptance Criteria

* The Today screen opens by default.
* The current task is displayed at the top.
* The next scheduled task is visible.
* All remaining tasks are listed chronologically.
* Completed tasks are clearly marked.
* Overdue tasks are highlighted.

---

# Epic 2 – Complete a Task

## User Story

As a family member,

I want to quickly mark a task as complete,

so everyone knows it has been done.

### Acceptance Criteria

* User taps the completion checkbox.
* A list of family members appears.
* Selecting a name completes the task.
* Timestamp is recorded.
* Progress updates immediately.
* Leaderboard updates immediately.
* Success animation is displayed.
* Success sound plays.

---

# Epic 3 – View Daily Progress

## User Story

As a family member,

I want to see how much of today's schedule has been completed,

so I know what remains.

### Acceptance Criteria

* Progress is shown as a percentage.
* Completed task count is displayed.
* Remaining task count is displayed.
* Progress updates instantly.
* 100% completion triggers celebration animation.

---

# Epic 4 – View the Leaderboard

## User Story

As a family member,

I want to compare points with the rest of the family,

so participation is fun and encouraging.

### Acceptance Criteria

* Daily leaderboard is available.
* Weekly leaderboard is available.
* Monthly leaderboard is available.
* All-Time leaderboard is available.
* Rankings update automatically.

---

# Epic 5 – View Task History

## User Story

As a family member,

I want to review previously completed tasks,

so I can see what has been done.

### Acceptance Criteria

* History displays completed tasks.
* History shows who completed the task.
* Completion timestamps are visible.
* Entries can be filtered.
* Search returns matching records.

---

# Epic 6 – Edit Incorrect Entries

## User Story

As an adult,

I want to correct accidental task completions,

so the history remains accurate.

### Acceptance Criteria

* Adults can edit completed tasks.
* Children cannot edit records.
* Leaderboards recalculate automatically.
* Daily progress updates when necessary.

---

# Epic 7 – Log a Potty Accident

## User Story

As a family member,

I want to quickly record a potty accident,

so we can monitor Mizu's progress.

### Acceptance Criteria

* Logging takes less than five seconds.
* Date and time default automatically.
* Pee or Poop can be selected.
* Notes are optional.
* Photos are optional.
* Entry appears immediately in History.

---

# Epic 8 – Add a Growth Journal Entry

## User Story

As a family member,

I want to record milestones,

so we have a lasting record of Mizu's development.

### Acceptance Criteria

* Users can create journal entries.
* Photos can be attached.
* Weight can be recorded.
* Milestones appear chronologically.
* Entries can be filtered by type.

---

# Epic 9 – Receive Reminders

## User Story

As a family member,

I want reminders when tasks become due,

so Mizu's routine stays consistent.

### Acceptance Criteria

* Notifications occur at scheduled times.
* Reminder sounds play.
* Browser notifications appear when enabled.
* Notifications respect user settings.

---

# Epic 10 – Complete Recurring Care

## User Story

As an adult,

I want recurring care reminders,

so grooming and hygiene tasks are never forgotten.

### Acceptance Criteria

* Bath reminders appear automatically.
* Nail grinding reminders appear automatically.
* Completing the reminder schedules the next occurrence.
* Recurring reminders appear on the Today screen.

---

# Epic 11 – Continue Working Offline

## User Story

As a family member,

I want to continue using the app without internet,

so I can still care for Mizu anywhere.

### Acceptance Criteria

* Today's schedule remains available.
* Tasks can still be completed.
* Potty accidents can be logged.
* Journal entries can be created.
* Data synchronizes automatically once online.

---

# Epic 12 – Celebrate Daily Success

## User Story

As a family,

We want completing every task to feel rewarding,

so everyone enjoys using the app.

### Acceptance Criteria

* 100% completion displays confetti.
* Celebration message is shown.
* Success sound plays.
* Daily progress displays 100%.
* All tasks show completed status.
